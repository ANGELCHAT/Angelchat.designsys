### Buttons Wrapper

Use it when you need to add spacing between inlined buttons.

```js
<ButtonsWrapper>
  <Button primary>Primary Button</Button>
  <Button secondary>Secondary Action</Button>
</ButtonsWrapper>
```

```js noeditor
<ComponentHtmlMarkup>
  <ButtonsWrapper>
    <Button primary>Primary Button</Button>
    <Button secondary>Secondary Action</Button>
  </ButtonsWrapper>
</ComponentHtmlMarkup>
```
